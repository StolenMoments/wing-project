# wing-project

인디 음악 플랫폼 Win:G

- 패키지 관리는 **yarn** (https://medium.com/@ehddnjs8989/npm-vs-yarn-3a611c89d291)
  - 프로젝트 실행은 **yarn start**
  - 패키지 설치는 **yarn add -패키지명-**
  

- 초기 환경에 추가한 패키지
  - react-router-dom
  - @material-ui/core
  - bootstrap
  - reactstrap
  - storybook
  - @storybook/addon-knobs
  - styled-components
